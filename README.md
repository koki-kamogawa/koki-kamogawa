## 主なプロジェクト

### ⭐ AI Agent Team Template — 10人のAIチーム構築テンプレート
> Claude Codeで10人の専門AIエージェントチームを構築するためのオープンソーステンプレート

- 秘書・企画・マーケ・監査・営業・自動化設計・開発・調査・文書・リーダーの10エージェント
- 品質管理hooks・メモリシステム・handoff規約付き
- 実際の事業運営で使用中
- **[→ リポジトリを見る](https://github.com/koki-kamogawa/ai-agent-team-template)**

---

### Lumo — 音声日記 + AI分析
> 声で日記をつけると、AIが感情や生活パターンを分析してくれる

- 音声入力でリアルタイム文字起こし
- AIによるムードトラッキングと週次レポート
- Next.js / Whisper API / Claude API
- **[→ リポジトリを見る](https://github.com/koki-kamogawa/Lumo)**

---

### Nebula Notes — 音声メモ → AI要約
> 声でメモを取るだけで、AIが自動で整理・要約してくれる

- 音声認識による自動文字起こし
- AIによる要約・タグ付け・検索
- Next.js / TypeScript / Tailwind CSS
- **[→ リポジトリを見る](https://github.com/Oovo-project/nebula-notes)**

---

### zuretta — パーティーゲームWebアプリ
> スマホ1台を回して遊ぶ、秘密回答型パーティーゲーム

- バックエンド不要。ブラウザだけで動く
- 200問以上の内蔵お題、モバイルファースト設計
- Next.js / TypeScript / Tailwind CSS
- **[→ リポジトリを見る](https://github.com/Oovo-project/zuretta)**
